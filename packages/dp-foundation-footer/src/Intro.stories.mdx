# Foundation

Great minds global Footer

## Install

```
npm install @greatminds/dp-foundation-footer@latest
```

## Use

Import components you want into your UI

`import { Footer } from '@greatminds/dp-foundation-footer';`

and use them like so

```
const example = () => (
    <div>
        <Footer />
    </div>
)

```
